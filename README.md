# Felix Hernandez Vieyra - Resume

## Quick Start

```bash
# Make the build script executable
chmod +x build.sh

# Build the PDF
./build.sh
```

This will generate `<resume>.pdf` in the same directory.

## Workflow

1. **Edit** `<resume>.tex` with your changes
2. **Build locally** using `./build.sh` to test
3. **Commit and push** to GitHub
4. **GitHub Actions** automatically compiles and commits the PDF
5. **Download** the PDF from the repository
